# Windlab Fitness 💪

Uma aplicação de e-commerce moderna para produtos e serviços de fitness, desenvolvida com React, TypeScript e Vite.

## 🎯 Sobre o Projeto

Windlab Fitness é uma plataforma web completa para compra e venda de produtos fitness. O projeto oferece uma experiência de usuário intuitiva com funcionalidades robustas para clientes e administradores.

### Recursos Principais

- ✅ **Catálogo de Produtos**: Navegue por uma variedade de produtos fitness organizados por categorias
- 🛒 **Carrinho de Compras**: Adicione, remova e gerencie seus produtos
- 👤 **Autenticação de Usuários**: Sistema seguro de login e cadastro
- 👨‍💼 **Dashboard Admin**: Painel para gerenciar produtos e pedidos
- 🔍 **Busca e Filtros**: Encontre produtos facilmente
- 📱 **Design Responsivo**: Interface otimizada para todos os dispositivos
- 💾 **Persistência Local**: Carrinho salvo no navegador

## 🛠️ Tecnologias Utilizadas

- **Frontend**: React 19, TypeScript, Tailwind CSS, Vite
- **Backend**: Node.js, Express
- **UI Components**: Lucide React (ícones), Motion (animações)
- **AI Integration**: Google GenAI
- **Ferramentas**: esbuild, Autoprefixer

## 📦 Instalação

### Pré-requisitos
- Node.js (v18+)
- npm ou yarn

### Setup

1. Clone o repositório:
```bash
git clone https://github.com/Adilson-buca45/windlab-fitness.git
cd windlab-fitness
```

2. Instale as dependências:
```bash
npm install
```

3. Configure as variáveis de ambiente (se necessário):
```bash
cp .env.example .env.local
```

4. Inicie o servidor de desenvolvimento:
```bash
npm run dev
```

5. Acesse a aplicação:
```
http://localhost:5173
```

## 🚀 Scripts Disponíveis

- `npm run dev` - Inicia o servidor de desenvolvimento
- `npm run build` - Compila o projeto para produção
- `npm start` - Inicia o servidor em produção
- `npm run lint` - Verifica tipos TypeScript
- `npm run clean` - Remove arquivos de build
